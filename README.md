# Piyush Kumar -- UI/UX Portfolio

A modern, responsive portfolio showcasing my work as a UI/UX Designer
focused on complex dashboards, scalable design systems, and high-clarity
product experiences.

## Live Demo

 https://piyush-ux-portfolio.vercel.app/

## Features

-   Modern dark UI with glassmorphism aesthetics\
-   Fully responsive and mobile-first design\
-   Smooth animations using Framer Motion\
-   Case study--driven project showcase\
-   Working contact form (EmailJS integration)\
-   Optimized for performance and Vercel deployment

## Tech Stack

-   React (Vite)
-   Tailwind CSS
-   Framer Motion
-   EmailJS
-   Vercel
